# Content Maker Agent

Веб-приложение для генерации постов в Telegram-каналы на основе локальной LLM (Ollama). Работает без интернета и VPN — модель запускается на сервере.

## Каналы

| ID | Название | Тематика |
|---|---|---|
| `mice_japan` | MICE Japan | MICE-туризм в Японию: отели, гастрономия, культура, логистика групп |
| `match_point` | MATCH POINT | MICE в России и мире: тренды, новости рынка, обзоры площадок и отелей |
| `turdezhur` | ТурДежурный | AI-решения для отелей и турфирм |

Автор всех каналов — Ольга Харлёнок. Голос единый, подача разная под аудиторию.

## Стек

- **Backend:** Python / Flask
- **LLM:** Ollama (локально) — модель `llama3.1:8b` по умолчанию
- **Frontend:** Vanilla HTML/CSS/JS, тёмная тема
- **Промпты:** `prompts.py` — system prompt + few-shot примеры из `examples.json`

## Структура файлов

```
app.py              — Flask-сервер, роуты /generate и /health
prompts.py          — промпты и характеры каналов, грузит примеры из examples.json
examples.json       — реальные посты каналов (few-shot обучение), парсится из docx
templates/
  index.html        — весь фронтенд
Match Point/
  posts Match Point.docx   — исходные посты канала
MICE Japan/
  Posts Mice Japan.docx    — исходные посты канала
Turdezhur/          — папка под будущие посты канала
START.md            — инструкция по установке и запуску
requirements.txt    — flask, requests
```

## Запуск

```bash
# 1. Ollama (в отдельном терминале)
ollama serve

# 2. Приложение
python app.py

# 3. Браузер
http://localhost:5000
```

## Переменные окружения

| Переменная | По умолчанию | Описание |
|---|---|---|
| `OLLAMA_URL` | `http://localhost:11434` | Адрес Ollama |
| `OLLAMA_MODEL` | `llama3.1:8b` | Модель. Для слабых машин: `llama3.2:3b` |

## Как добавить примеры постов

1. Положить `.docx` с постами в папку канала
2. Запустить парсер:
```bash
python parse_posts.py
```
Это перезапишет `examples.json` — промпты подхватят автоматически.

> Пока парсер встроен в одноразовый скрипт в истории сессии. Вынести в `parse_posts.py` при необходимости.

## Деплой на российский VPS

Те же шаги что и локально. Рекомендуемые хостинги: Timeweb, Selectel, Яндекс Облако.
Минимальные требования сервера: 16 GB RAM, 30 GB диск, 4 CPU.
После запуска доступен по IP без VPN.

## Известные ограничения

- Для канала **ТурДежурный** примеры постов ещё не добавлены — нужно положить docx в папку `Turdezhur/`
- Модель `llama3.1:8b` пишет на русском хорошо, но не идеально — качество регулируется промптами в `prompts.py`
- Длинные посты генерируются ~30-60 секунд на CPU без GPU
