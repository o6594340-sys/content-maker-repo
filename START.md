# Запуск Content Maker

## Первый раз

### 1. Установить Ollama
Скачать: https://ollama.com/download
Запустить установщик, перезапустить терминал.

### 2. Скачать модель (один раз)
```
ollama pull llama3.1:8b
```
Займёт 5-10 минут, ~5 GB.

### 3. Установить зависимости Python
```
pip install -r requirements.txt
```

---

## Каждый раз

### Шаг 1 — запустить Ollama (в отдельном терминале)
```
ollama serve
```

### Шаг 2 — запустить приложение
```
python app.py
```

### Шаг 3 — открыть в браузере
```
http://localhost:5000
```

---

## Если мало RAM (меньше 16 GB)

Используй меньшую модель:
```
ollama pull llama3.2:3b
```

И измени в app.py строку:
```python
OLLAMA_MODEL = os.getenv("OLLAMA_MODEL", "llama3.2:3b")
```

---

## Для доступа с других компьютеров в сети

Узнай свой IP:
```
ipconfig
```
Другие пользователи открывают: `http://твой-IP:5000`
